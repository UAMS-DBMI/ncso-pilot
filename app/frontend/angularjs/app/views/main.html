NCSO Demo

This demo shows how the National Children Study Ontology (NCSO) is used to query data that is structured using the MDES.

It uses a synthetic data set because we didn't have enough data about the phenomena we looked at in the existing data. However, the whole structure of the demo is such that the methodology can be used on real MDES structured data without major adjustments.

It also stresses how the ontology can help to integrate data about phenomena of the same kind captured by different instruments and shows how maintaining and querying the data can be kept flexible across version changes. In addition, we demonstrate how the use of data items in a larger context (for example general health surrogate data) is supported. This will enable studies on the utility of the data collected by the NCS.
